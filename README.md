## الكتب المتاحة
1. [نحو تدبر واعٍ - النسخة العربية](books/conscious-contemplation/ar)

## المميزات
- واجهة ثنائية اللغة (عربي/إنجليزي)
- تصميم متجاوب

- تنظيم واضح

## كيفية الاستخدام

3. استخدم أزرار التبديل بين اللغات
